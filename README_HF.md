# 🎵 网易云音乐 24 小时挂机

## 使用说明

1. 在下方控制面板输入您的 MUSIC_U Cookie
2. 输入要播放的歌单 ID
3. 点击"开始播放"按钮

## 获取 MUSIC_U Cookie

1. 访问 https://music.163.com
2. 登录账号
3. 按 F12 打开开发者工具
4. 在 Network 标签页找到任意请求
5. 复制 Cookie 中的 MUSIC_U 值

## 获取歌单 ID

打开任意歌单，复制 URL 中的数字部分
例如：https://music.163.com/#/playlist?id=1959142287
歌单 ID 为：1959142287

## ⚠️ 注意事项

- 请妥善保管您的 Cookie，不要泄露
- 合理使用挂机功能
- 本工具仅供学习交流使用
